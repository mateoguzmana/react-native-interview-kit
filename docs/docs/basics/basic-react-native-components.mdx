# Basic React Native Components

## What are the basic React Native components? Explain their usage.

This question might come around to see if you know the fundamentals.

Short answer: View, Text, Image, ScrollView, TextInput, StyleSheet.

You can learn more about them on the [official docs](https://reactnative.dev/docs/components-and-apis#basic-components).
